🍔 Maceolands

Maceolands is a simple food ordering website built using HTML, CSS, and JavaScript.

🚀 Features
	•	Interactive menu display
	•	Add items to cart
	•	Automatic cart total calculation
	•	Simple AI assistant (simulated responses)

🛠️ Technologies Used
	•	HTML
	•	CSS
	•	JavaScript

📦 How to Use
	1.	Open the website
	2.	Browse the menu
	3.	Click Add to add items to your cart
	4.	View your total price
	5.	Ask the AI assistant for food suggestions
